
# DuşakabinPro Web Uygulaması Kurulum Rehberi

Projeyi tamamladık. Bu web uygulaması Next.js ile geliştirilmiş, modern, hızlı ve responsive (mobil uyumlu) bir yapıdadır.

## Nasıl Çalıştırılır? (Windows)

1. **Terminali Açın** (VS Code terminali veya CMD).
2. Proje klasörüne gidin:
   ```powershell
   cd c:\Users\user\Desktop\merve_site\dusakabin-app
   ```
3. **Gerekli Paketleri Yükleyin** (Eğer yapmadıysanız):
   ```powershell
   npm install
   ```
4. **Uygulamayı Başlatın**:
   ```powershell
   npm run dev
   ```
5. Tarayıcınızı açın ve **http://localhost:3000** adresine gidin.

## Özellikler

- **Admin Paneli & Dashboard**: Sipariş, Üretim ve Stok özeti.
- **Sipariş Oluşturma**: En/Boy girildiğinde otomatik kesim listesi (Cam, Profil) hesaplar.
- **Kanban Üretim Hattı**: Siparişleri sürükle-bırak mantığıyla (veya butonla) Bekliyor -> İmalat -> Hazır aşamalarına taşıyın.
- **Stok Takibi**: Yapılan üretimler ve manuel girişlerle stoğu yönetin.
- **Yazdırılabilir Sipariş Fişi**: Sipariş detay sayfasında "Yazdır" butonu.
- **Mobil Uyumlu**: Telefonunuzdan da rahatça kullanabilirsiniz.

## Yayına Alma (Hosting)

- **Vercel**: En kolay yöntem. Projeyi GitHub'a yükleyip Vercel'e bağlarsanız ücretsiz SSL ile yayına alabilirsiniz.
- **Hostinger (VPS)**: Node.js desteği olan bir sunucuda `npm run build` ve `npm start` ile çalıştırabilirsiniz.
- **Veritabanı**: Şu an veriler tarayıcıda (LocalStorage) tutulmaktadır. Demo ve tek bilgisayar kullanımı için uygundur. Verilerin silinmemesi veya ortak kullanım için Firebase bağlantısı `lib/db.ts` dosyası üzerinden yapılabilir.
